## Steps to Run

#### Step 01:

Clone the git repository:

```
git clone https://github.com/SLDrago/Calculator.git
```

#### Step 02:

Go to the Calculator folder just downloaded:

```
cd Calculator
```

#### Step 03:

Install the dependecies:

```
npm install
```

#### Step 02:

Run the application:

```
npm run dev
```

## For Build

Run this command:

```
npm run build
```

preview:

```
npm run preview
```
